## INSTRUCTIONS
1. Read the Search Data provided above.
2. Filter out any information that does not answer the User Question.
3. Ignore ads, navigation menus, cookie banners, and unrelated news.
4. Extract key facts, dates, numbers, and direct quotes relevant to the question.
5. Structure the output as a clean bulleted list.
6. Preserve source discipline:
   - Include the source name or URL when the search data provides it.
   - If SOURCE COVERAGE says there was only one readable source, say the findings are based on one readable source.
   - If only snippets were available, do not infer details beyond the snippets.
   - Do not turn a single-source result into a broad "latest news" survey.

## OUTPUT FORMAT
- Use bullet points.
- Be factual and objective.
- If the data does not contain the answer, state that clearly.
